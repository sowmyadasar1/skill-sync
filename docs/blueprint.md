# **App Name**: SkillSync

## Core Features:

- Event Feed: Display a feed of upcoming developer events such as workshops, hackathons, and speaker sessions.
- Ask Gemini: Integrate Gemini AI to answer development-related queries; Gemini will be used as a tool.
- Collaboration Board: Provide a platform for students to post or join projects and collaborations.
- Resource Navigation: Use Gemini to suggest relevant learning resources based on user interests, serving as a tool to decide the usefulness of including certain suggested resources.
- User Profiles: Enable users to create and manage profiles with their tech interests and skills.
- Saving Items: Allow users to save events and resources for future reference.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to represent technology and connection.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Yellow-orange (#F2A333) to highlight important information and CTAs.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clean and modern icons to represent different event categories and resources.
- Maintain a clean and organized layout for easy navigation and user experience.
- Subtle transitions and animations to enhance user engagement and provide visual feedback.
